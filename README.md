# Pickul

Friend preference comparison.

## Features

1. **Create Collections** - Upload a group of related images into a collection
2. **Rank by Preference** - Drag-and-drop or compare pairs to order images by your personal preference
3. **Share with Friends** - Send a link to friends so they can create their own rankings
4. **Compare Results** - View side-by-side comparisons highlighting:
   - Common patterns (images everyone ranked similarly)
   - Outliers (where preferences diverged)

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS v4
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
